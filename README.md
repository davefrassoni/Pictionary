# Pictionary #
current version: 0.1.2

## Description ##

### Enjoyable Pictionary game working with Node.js and Socket.io ###

## Installation ##

### Either download from GitHub or run: ###

		npm install Pictionary

## Setup ##

1. Run:

		node server.js

1. Browse to:

		http://localhost:8080/

1. Enjoy the Pictionary!

1. Modify code as you want
